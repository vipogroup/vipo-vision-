💻 התקנה מלאה על המחשב

ודא שמותקנים אצלך: Node.js, Git, FFmpeg

  1. פתח PowerShell כ-Administrator (קליק ימני → Run as Administrator)
  2. העתק והדבק את הפקודה הבאה:
powershell -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/vipogroup/vipo-vision-/main/install.ps1 | iex"
  1. ההתקנה תרוץ אוטומטית — ייקח כ-2 דקות
  2. בסיום, הדשבורד ייפתח אוטומטית בדפדפן
  3. המערכת תעלה אוטומטית עם כל הפעלה של המחשב

💡 אחרי ההתקנה, גש ל: http://localhost:5055

📺 צפייה מהדפדפן

אם המערכת כבר מותקנת על מחשב ברשת שלך, פשוט פתח בדפדפן:

http://<IP-של-המחשב>:5055

לדוגמה: http://10.0.0.4:5055

בקש מהאדם שהתקין את המערכת את ה-IP שלו. עובד מכל מכשיר ברשת — מחשב, טלפון, טאבלט.

💡 הדפדפן יציע לך גם "להתקין כאפליקציה" — זה PWA שעובד גם בלי אינטרנט.
